# Реестр репозиториев

> **Source-of-truth** для списка репозиториев вашей экосистемы.
> Обновляется при создании/удалении репозиториев.

## Типы репозиториев (4 типа)

| Тип | Критерий | Source-of-truth |
|-----|----------|-----------------|
| **Pack** | Source-of-truth области | Да |
| **Framework** | Рамки корректности (FPF/SPF) | Да |
| **Format** | Протокол оформления | Да (для формата) |
| **Downstream** | Производные от Pack | Нет |

## Репозитории

| # | Репозиторий | Тип | Содержание | Статус |
|---|-------------|-----|------------|--------|
| 1 | exocortex-template | Format | Шаблон экзокортекса | Active |
| 2 | my-strategy | Downstream/governance | Стратегический хаб | Active |
| 3 | strategist-agent | Downstream/instrument | Агент Стратег | Active |

> Добавляйте ваши репозитории по мере создания.

## Правила

1. **Pack — единственный source-of-truth**
2. **Один репозиторий — один тип**
3. **При добавлении** — обновить этот реестр
4. **REPO-TYPE.md обязателен** для всех репозиториев
