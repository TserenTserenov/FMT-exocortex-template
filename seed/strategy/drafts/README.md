# Черновики (Drafts)

> **Тип:** Личные черновики — тексты из отсортированных заметок, никому не показываем.
> **Индекс:** `drafts/draft-list.md` — навигируемый список всех черновиков.
> **TTL:** 7 дней. На стратегировании: приоритизировать, продвинуть или закрыть.

## Формат файла

```markdown
---
source_notes: [fleeting-notes YYYY-MM-DD #N, #M]
target: заготовка | pack | рп
created: YYYY-MM-DD
ttl: YYYY-MM-DD
---

# Название

[связный текст из заметок]
```

## Жизненный цикл

```
Заметка (fleeting-notes.md)
    ↓ Note-Review → категория «Черновик»
Черновик (drafts/*.md)
    ↓ Работа: обогащение, связывание с Pack
    ├── → captures.md (знание извлечено → Pack)
    ├── → Индекс знаний status: draft (заготовка)
    ├── → РП в WeekPlan (задача)
    └── → archive (осознанно закрыт)
```

## Правило незастревания

Черновик старше 7 дней = вопрос на стратегировании:
1. Приоритизировать → в «Приоритетные» в draft-list.md
2. Продвинуть → превратить в заготовку или captures
3. Закрыть → archive/notes/ с причиной
