# Draft List (Список черновиков)

> **Материализация:** PD.WP.011 Draft List (PACK-personal)
> **Обновляется:** Note-Review (ежедневно) + Strategy Session (еженедельно)
> **Anti-accumulation:** TTL 7 дней. Ревью на каждом стратегировании.
> **Правило:** черновик — ЛИЧНЫЙ текст. Никому не показываем. Показываем = заготовка (Индекс знаний, status: draft).

## Конвейер (4 стадии артефакта)

```
Заметка (7д) → Черновик (7д) → Заготовка (14д) → Пост
fleeting-notes   drafts/          Индекс знаний     Published
                 draft-list.md    status: draft       status: published
```

Каждый артефакт ОБЯЗАН выйти в один из пунктов за TTL:

| Стадия | TTL | Выходы (один из) |
|--------|-----|-----------------|
| Заметка | 7 дней | captures / черновик / РП / archive |
| Черновик | 7 дней | captures / заготовка / РП / archive |
| Заготовка | 14 дней | пост (published) / archive |
