# Тип репозитория

**Тип**: `Format`
**Система (SoI)**: cross-cutting
**Содержание**: text-description
**Для кого**: public
**Source-of-truth**: yes (для формата экзокортекса)

## Upstream dependencies

- FPF (First Principles Framework)
- SPF (Second Principles Framework)

## Downstream outputs

- Форки пользователей (персональные экзокортексы)
- exocortex-setup-agent (агент развёртывания)

## Non-goals

- Хранение предметных знаний (это Pack)
- Хранение личных данных пользователя (это user-space после форка)
- Код ботов и сервисов (это отдельные downstream-репо)
